## 命题与命题联结词

* 否定：设 $P$ 为命题，$P$ 的否定是一个复合命题，记作：$not P$。若 $P$ 为 $T$， $not P$ 为 $F$；若 $P$ 为 $F$，$not P$ 为 $T$。
* 合取：设 $P$、$Q$ 为两个命题，$P$ 和 $Q$ 的合取是一个复合命题，记作 $P ^^ Q$。当且仅当 $P$、$Q$ 同时为 $T$ 时，$P ^^ Q$ 为 $T$，其余情况 $P ^^ Q$ 为 $F$。
* 析取：设 $P$、$Q$ 为两个命题，$P$ 和 $Q$ 的析取是一个复合命题，记作 $P vv Q$。当且仅当 $P$、$Q$ 同时为 $F$ 时，$P vv Q$ 为 $F$，其余情况 $P vv Q$ 为 $T$。
* 条件：设 $P$、$Q$ 为两个命题，$P$ 和 $Q$ 组成的条件命题是一个复合命题，记作 $P rarr Q$。当且仅当 $P$ 的真值为 $T$、$Q$ 的真值为 $F$ 时，$P rarr Q$ 为 $F$，其余情况 $P rarr Q$ 为 $T$。
* 双条件：设 $P$、$Q$ 为两个命题，$P$ 和 $Q$ 组成的双条件命题是一个复合命题，记作 $P harr Q$。当 $P$ 与 $Q$ 的真值相同时，$P harr Q$ 的真值为 $T$，否则 $P harr Q$ 的真值为 $F$。

**同或（$P vv Q$）——表示想容，描述的两个命题并不排斥，可以同时发生。**

**异或（$( P vv Q ) ^^ not ( P ^^ Q )$、$( P ^ not Q ) vv ( not P ^^ Q )$）——表示相斥，描述的两个命题不可以同时发生。**

## 命题公式的等值演算

命题逻辑中规定，联结词优先次序依次为：$not$，$^^$，$vv$，$rarr$，$harr$。

> 重要的命题定律

名称|公式
--|--
吸收律| $A vv ( A ^^ B ) <=> A$ $A ^^ ( A vv B ) <=> A$
德摩根律| $not ( A vv B ) <=> not A ^^ not B$ $not ( A ^^ B ) <=> not A vv not B$
蕴涵等值式| $A -> B <=> not A vv B$
等价等值式| $A <-> B <=> ( A -> B) ^^ ( B -> A)$
假言易位| $A -> B <=> not B -> not A$
归谬论| $( A -> B ) ^^ ( A -> not B ) <=> not A$

> 推理定律

名称|公式
--|--
假言推理| $P ^^ ( P -> Q ) => Q$
拒取式| $( P -> Q ) ^^ not Q => not P$
析取三段论| $( P vv Q ) ^^ not Q => P$
条件三段论| $( P -> Q ) ^^ ( Q -> R ) => ( P -> Q )$

${ not, vv }$、${ not, ^^ }$ 最小联结词完备集。
