# Windows 10 专业版与 Office 365 激活

> * 擅用 **谷歌**，弃用 ~~百度~~。

## 摘要
总的来说 ***Windows 10 专业版*** 激活比较方便，而 ***Office 365*** 激活前需要先转换版本。

## 前言
我苦于激活问题，遂查询资料后整理出一套完整激活流程。

## 准备

Office 专业增强版 2016 | XQNVK-8JYDB-WJ9W3-YJ8YR-WFG99

* 注：本文撰写时（2019年11月25日） ***Office 365*** 使用的是 Office16 版本也就是 ***Office 2016*** 实体。

## 激活 Windows 专业版

1. 卸载当前 Windows 版本的产品密钥
```bat
slmgr.vbs /upk
```
2. 安装 5×5 产品密钥
```bat
slmgr.vbs /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
```
3. 指定主机
```bat
slmgr.vbs /skms kms.03k.org
```
4. 联机激活
```bat
slmgr.vbs /ato
```

## 激活 Office 365

1. 转至 ospp.vbs 文件夹
```bat
 cd C:\Program Files\Microsoft Office\Office16
```
2. 将 Office 365 默认安装的家庭版转为批量版
```bat
 for /f %x in ('dir /b ..\root\Licenses16\proplusvl_kms*.xrm-ms') do cscript ospp.vbs /inslic:"..\root\Licenses16\%x"
 for /f %x in ('dir /b ..\root\Licenses16\proplusvl_mak*.xrm-ms') do cscript ospp.vbs /inslic:"..\root\Licenses16\%x"
```
3. 安装产品密钥
```bat
 cscript ospp.vbs /inpkey:XQNVK-8JYDB-WJ9W3-YJ8YR-WFG99
```
4. 设置 KMS 主机名
```bat
 cscript ospp.vbs /sethst:kms.03k.org
```
5. 激活已安装的 Office 产品密钥
```bat
 cscript ospp.vbs /act
```
